# Nifty Options Autonomous Trading System

An automated trading system for Nifty weekly options using technical analysis, institutional flow analysis, and machine learning.

## Features

- **Multi-timeframe Technical Analysis**: Analyzes Nifty data across multiple timeframes
- **Institutional Flow Analysis**: Incorporates FII/DII flows and smart money patterns
- **Dynamic Risk Management**: Adaptive position sizing and risk controls
- **Telegram Alerts**: Real-time notifications for signals and system status
- **Self-Learning**: Auto-optimization based on performance feedback
- **Cloud Deployment**: Ready for Render.com deployment

## Prerequisites

- Zerodha Kite Connect API access
- Telegram Bot Token
- Render.com account (optional)
